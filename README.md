# 🎲 Dice Game

A simple and fun **Dice Game** built using **HTML, CSS, and JavaScript**.  
Click on the **Shuffle button** to roll the dice. The player with the higher number wins! 🚩

---

## ✨ Features

- 🎨 Modern UI with gradient background and hover effects
- 🎲 Random dice images generated for Player 1 and Player 2
- 🏆 Winner is declared dynamically on screen
- ⚡ Lightweight and responsive design

## 🚀 How to Play

1. Open the game in your browser.
2. Click the **Shuffle** button.
3. Both dice roll and show random values.
4. The player with the higher number wins.
5. If both dice are the same → it's a tie! 😃
